# Repository Meta Skills

This repository is the `skyrim-mod-chs-translation` Codex plugin source for Windows-based The Elder Scrolls V: Skyrim SE/AE Simplified Chinese Mod localization.

The authoritative runtime plugin Skills live in `skills/` at the project root.

This `.codex/skills/` directory is intentionally limited to repository meta Skills:

- `skyrim-mod-chs-install`
- `skyrim-mod-chs-usage`
- `skyrim-mod-chs-maintenance`

Do not copy runtime translation, routing, QA, or final_mod assembly Skills back into this directory.
