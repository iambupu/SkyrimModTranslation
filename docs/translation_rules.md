# Translation Rules

## 语言

- 使用简体中文。
- 保持自然游戏本地化风格。
- 不要机翻腔。
- 不要把奇幻文本翻成现代网络用语。
- 不要擅自改写剧情含义。

## 类型规则

- UI/MCM 文本：短而准确。
- 书籍文本：可以更文学，但不能改变信息。
- 对话文本：符合角色身份。
- 任务目标：清晰直接。
- 物品名：短，像游戏内名称。
- 法术名：保留奇幻感。
- 技能/效果说明：准确优先。

## 禁止翻译

- FormID
- EditorID
- 脚本名
- 变量名
- 路径
- 文件名
- 插件名
- JSON key
- XML/HTML tag

## 必须保留

- 占位符
- 换行符
- 颜色标签
- 字体标签
- HTML/XML 标签
- MCM 控制符
- Papyrus 变量标记

## 不确定术语

- 写入 `qa/unresolved_terms.md`。
- 不要擅自硬翻。
- 需要用户确认后再进入 `glossary/mod_terms.md`。
